Overview¶

exodus-gw (or “exodus gateway”) is a microservice for uploading and publishing content onto Red Hat’s CDN. It exposes an API for use by publishing tools.

  • For usage information on the exodus-gw API, see API Reference.

  • For more general information on the CDN architecture and how exodus-gw fits in, see Architecture.
